# SharePoint IT Help Desk Project

## Overview
This SharePoint Framework (SPFx) project provides an **IT Help Desk system** for internal users.  
It allows employees to submit tickets, track their status, and communicate with IT staff efficiently through SharePoint.

---

## Features
- **Ticket Submission**: Users can submit IT support requests through a user-friendly form.
- **Ticket Tracking**: View the status of submitted tickets (Open, In Progress, Closed).
- **Home Page Integration**: Quick access to ticket submission and recent tickets.
- **SharePoint Lists**: All tickets are stored in a SharePoint list for easy management.
- **Responsive Design**: Works on both desktop and mobile devices.
- **Notifications**: Optional email notifications for ticket updates (if configured).

---

## Prerequisites
- Node.js LTS version
- SharePoint Framework (SPFx) installed
- Office 365 / SharePoint Online access
- Gulp CLI installed globally (`npm install -g gulp-cli`)
